# Python CLI Expense Tracker

A simple **Command-Line Interface (CLI)** expense tracker built with **Python**.  
This project allows you to **add, view, edit, and delete expenses**, and **calculate total spending**. Expenses are stored in a **JSON file** (`data.json`) for persistence.

---

## Features

- Add a new expense with amount, category, description, and date  
- View all expenses with details  
- Edit or delete expenses  
- View total expenses  
- Store expenses in JSON for persistent storage  
- Easy-to-use CLI menu
